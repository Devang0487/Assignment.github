Q-17 What are the different tags to seperate sections of text ?

Ans-17 1)br Tag:-

This tag Seperate the Line of Text.

Break the current line and shift the text to new line.

2) P Tag:-

This tag Write a Paragraph of Text.

3)BlockQuote tag:-<

This tag is used to define large quoted section.
