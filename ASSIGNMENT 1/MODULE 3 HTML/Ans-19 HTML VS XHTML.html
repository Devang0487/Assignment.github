Q-19 What is Difference between Html and Xhtml ?

Ans-19

HTML:-

Html tags are not Extensible.

Html tags are not case sensitive.

Html tags are overlapping tags.

Html tag possible to leave off and ending tag.

Like:Body Tag

XHTML:-

XHtml tags are Extensible.

XHtml tags are only lowecase tags are allowed.

XHtml tags are not overlapping tags.

XHtml tags should appear in pairs.
