Ex:-Html Tag,Head Tag,Title Tag,Meta Tag,Body Tag,A Tag Img Tag,Paragraph Tag Etc...

Html Attributes Provides Additional Information About Element. Attributes are always specified in the start tag and usually come in Name/Value Pair. Like:-name="Value"
